This project is used as a reproducer to show the difference in behaviour in the generation of the identifier between atlas-identities 1.1.0 and 1.1.4

# Compile
```
$ mvn clean install
```

# Run
```
$ java -jar target/atlas-test-1.0-SNAPSHOT-jar-with-dependencies.jar
```

Update the atlas version in pom.xml to 1.1.3 or 1.1.4 and retry
